## Overview

My Own YouTube App is a simple and useful application where users can **save and watch YouTube videos inside the app**.

In this app:

* The user pastes a YouTube video link
* The video is saved in the database
* The user can watch the same video later directly from the app

---

## Core Features

## Add Video

* User can paste a YouTube video link
* User must enter:

  * Video Title
  * Video Link
* User selects a **category** before adding the video
* Video is saved in the database

---

## Category System

* Predefined categories are available
* User selects a category while adding a video
* Videos are organized category-wise

---

## Watch Videos

* All added videos are displayed in the app
* When user clicks on a **video thumbnail**, the video opens on the next screen
* User can watch the video directly inside the app

---

## Related Videos

* When a video is opened:

  * Related videos from the **same category** are shown below
* Similar to a mini YouTube experience

---

## Search & Filter

* Search videos by **video title**
* Filter videos by **category**

---

## Problem Solved

This project solves a real-world problem:

Example:
A user watches a lecture on YouTube but does not complete it. Later, the user has to search for the same video again on YouTube.

Solution with this app:

* User saves the video link once in the app
* Next time, the video is already available
* No need to search again
* User can directly continue watching from the app

---

## Data Storage

* All videos are stored in a **database**
* Each video includes:

  * Title
  * Link
  * Category

---

## User Flow

1. User opens the app
2. Adds a video (title + link + category)
3. Video is saved
4. User clicks on thumbnail
5. Video opens on next screen
6. Related videos appear below
